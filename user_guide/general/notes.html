Notes

BitAuth stores permissions in the session, which means if you are using CodeIgniter's default session settings, permissions are stored in a cookie, unencrypted. I can't advise against this strongly enough. You should, at a bare minimum be encrypting the session cookie by enabling sess_encrypt_cookie in config.php. Even better, you should be using database sessions so the information is stored server-side.
